# Excel to CSV
Script em Python para converter uma planilha Excel em um arquivo CSV.

## Requisitos
- Python 3.x
- pandas
- openpyxl

## Instalação
```bash
pip install pandas openpyxl
```

## Uso
```bash
python converter.py usuarios.xlsx
```
Ou adicionando um sufixo ao departamento:
```bash
python converter.py usuarios.xlsx AD
```
